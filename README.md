# Node.js Request Logger

A simple Node.js server built using the native `http` and `fs` modules.

## Features

- Basic HTTP server
- Custom routing
- Request logging
- File handling using `fs`
- Asynchronous callbacks

## Routes

| Route | Description |
|-------|-------------|
| `/` | Homepage |
| `/about` | About page |

## Installation

Clone the repository:

```bash
git clone <your-repo-link>
```

Move into the project folder:

```bash
cd node-request-logger
```

Install dependencies:

```bash
npm install
```

Run the server:

```bash
node index.js
```

## Server

The server runs on:

```txt
http://localhost:1506
```

## Technologies Used

- Node.js
- HTTP Module
- File System (`fs`) Module
